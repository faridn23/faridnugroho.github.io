BolaMania

BolaMania memberikan anda informasi seputar sepak bola yang ada di Eropa. BolaMania ini di bangun menggunakan framework Materialize Css dan juga dengan Menggunakan Progressive Web App

Website BolaMania ini dibangun menggunakan framework Materialize CSS dan menerapkan teknologi Progressive Web App atatu biasa disingkat dengan PWA. Kelebihan dari PWA adalah mengoptimasi sebuah website sehingga menjadi lebih cepat namun juga memberikan pengalaman bagi pengguna seperti sedang menggunakan aplikasi mobile.
